
# Sweet Delights — Final POE 


- index.html — main single-page application (Home + Menu Management views)
- style.css — styling
- app.ts — TypeScript source (included)
- app.js — JavaScript (browser-ready)
- README.md, CHANGELOG.md — submission notes

How to run:
1. Open `index.html` in a modern browser or use VSCode Live Server.
2. Use the Manager screen to add/remove menu items. The Home screen shows averages per course.
3. The app persists menu items using localStorage.
